# Original Technical Design

Esta carpeta conserva la documentacion original del diseno tecnico.

El archivo principal es [`technical-design-original.md`](technical-design-original.md).

## Regla de preservacion

Este documento representa el plan original y no debe sobrescribirse ni reescribirse para reflejar cambios posteriores de implementacion.

Si la arquitectura cambia, crear un documento nuevo que describa el estado final o la decision actual, y mantener este original como referencia historica.
